# Aplikasi Metode Numerik

Aplikasi web untuk mencari akar persamaan non-linear menggunakan metode Bisection dan Fixed Point Iteration.

## Teknologi

- React
- Vite
- Material UI
- Math.js
- Recharts

## Menjalankan Project

Install dependency:

```bash
npm install
```

Jalankan mode pengembangan:

```bash
npm run dev
```

Build untuk dibuka melalui Laragon:

```bash
npm run build
```

Setelah build selesai, buka folder project melalui Laragon. File `index.php` akan menyajikan hasil build dari folder `dist`.
