# Attributions

Project ini menggunakan komponen dan library open-source berikut:

- Material UI, digunakan sebagai komponen antarmuka.
- shadcn/ui dan Radix UI, digunakan sebagai referensi komponen antarmuka.
- Lucide React, digunakan untuk ikon.
- Recharts, digunakan untuk visualisasi grafik.
- Math.js, digunakan untuk evaluasi ekspresi matematika.
